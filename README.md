# 基於卷積神經網絡(CNN)之通用型水果新鮮度辨識系統

針對生鮮供應鏈品質篩選痛點，開發通用型電腦視覺影像分類模型，精準辨識3種水果的「新鮮 (Fresh)」與「腐爛 (Rotten)」共6個類別。

## 1.核心技術：
- TensorFlow / Keras / NumPy / Matplotlib
## 2.數據集規模：
- 總計13599張水果影像（訓練集10901張、測試集2698張）。
## 3.技術架構亮點：
- 數據增強技術：導入 RandomFlip、RandomRotation與RandomZoom，模擬產線相機鏡頭的不同視角與距離。
- 輕量化網絡設計：採用3層Conv2D結構（32➜64➜128），總參數僅482萬，模型大小為18.42MB，利於嵌入式產線設備的質檢推論部署。
- 防禦泛化陷阱：使用Dropout與EarlyStopping機制，最終驗證集準確率穩定達到92.4%。並規劃引入Grad-CAM與HSV色彩空間作為對抗環境反光斑點的優化方案。
